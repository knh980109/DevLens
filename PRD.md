# PRD - AI Code Review Dashboard (DevLens)

## 1. 문제 정의

개발팀은 코드 리뷰 현황, 품질 추이, 기여도를 한눈에 파악하기 어렵다.

**구체적인 페인 포인트:**
- PR이 얼마나 오래 열려 있는지, 리뷰에 평균 몇 시간이 걸리는지 파악 불가
- 어떤 개발자가 어떤 품질의 코드를 작성하는지 데이터 기반 평가 수단 없음
- 어떤 파일/모듈에서 이슈가 반복 발생하는지 패턴 분석 수단 없음
- 팀 전체의 코드 품질 트렌드가 개선되고 있는지 시각적 확인 불가

## 2. 목표

- 코드 리뷰 현황을 대시보드로 시각화하여 팀 생산성 가시화
- AI 분석 기반 코드 품질 점수 제공으로 객관적 기준 마련 (Mock)
- 개발자별 기여도 및 역량 지표를 레이더 차트로 직관적 제공
- 반복 발생하는 이슈 패턴을 AI 인사이트로 선제적 개선 유도

## 3. 성공 지표 (Success Metrics)

| 지표 | 목표값 |
|------|--------|
| 페이지 로드 시간 | < 2초 |
| API 응답 시간 | < 200ms |
| 평균 리뷰 소요시간 목표 | 24시간 이내 |
| 코드 품질 목표 점수 | 85점 이상 |
| 지원 해상도 | 768px ~ 1920px |

## 4. 핵심 기능

### 4.1 Overview 페이지 (메인 대시보드)
- 6개 KPI 카드: 전체·오픈·머지·클로즈 PR 수, 평균 품질점수, 평균 리뷰시간
- 카운터 애니메이션: 페이지 진입 시 0 → 목표값 easing 애니메이션
- 주간 PR 추이: 전체 PR vs 머지 PR 11주 Area 차트
- 코드 품질 트렌드: 주간 품질점수 라인 차트 (목표 기준선 표시)
- 개발자별 PR 현황: 전체 vs 머지 Grouped Bar 차트
- 평균 리뷰 소요시간: RadialBar 게이지 차트

### 4.2 PR 목록 페이지
- 상태별 필터 탭: All / Open / Merged / Closed (건수 뱃지 포함)
- PR 목록 테이블: 제목, 작성자, 리뷰어, 품질점수 배지, 상태 배지, 날짜
- 품질점수 색상 기준: 90↑ 초록 / 70~89 노랑 / 70↓ 빨강
- 슬라이드 디테일 패널: 우측 슬라이드인, AI 품질 분석 요약 포함
- 패널 닫기: ESC 키 / 오버레이 클릭

### 4.3 개발자 분석 페이지
- 개발자 카드 그리드: 이름, 역할, 총 PR, 머지율, 평균 품질점수
- 카드 hover 애니메이션: translateY lift 효과
- 상세 모달: 레이더 차트 (속도/품질/협업/일관성/커버리지 5축) + 통계 수치

### 4.4 AI 인사이트 페이지
- 심각도 필터: All / Critical / Warning / Info
- 인사이트 카드: 심각도 아이콘·색상, 카테고리 배지, 파일명·라인번호, 개선 제안
- 스태거 애니메이션: 카드 순차 fade-in (80ms 간격)
