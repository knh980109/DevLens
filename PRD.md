# PRD - AI Code Review Dashboard

## 1. 문제 정의
개발팀은 코드 리뷰 현황, 품질 추이, 기여도를 한눈에 파악하기 어렵다.
PR마다 리뷰가 얼마나 걸리는지, 어떤 개발자가 어떤 품질의 코드를 작성하는지,
어떤 파일/모듈에서 이슈가 자주 발생하는지 가시화된 도구가 없다.

## 2. 목표
- 코드 리뷰 현황을 실시간 대시보드로 시각화
- AI 분석 기반 코드 품질 점수 제공 (Mock)
- 개발자별 / 레포지토리별 기여도 및 이슈 패턴 파악

## 3. 핵심 기능

### 3.1 Overview 페이지 (메인 대시보드)
- 전체 PR 수, 오픈/머지/클로즈 현황 카드 (카운터 애니메이션)
- 주간 PR 추이 라인 차트
- 평균 리뷰 소요시간
- 코드 품질 점수 게이지

### 3.2 PR 목록 페이지
- PR 목록 테이블 (상태, 작성자, 리뷰어, 생성일, 품질점수)
- 상태별 필터 (Open / Merged / Closed)
- PR 클릭 시 상세 슬라이드 패널

### 3.3 개발자 분석 페이지
- 개발자별 PR 수, 머지율, 평균 품질점수 랭킹
- 개발자 클릭 시 상세 레이더 차트 (속도/품질/협업/일관성)

### 3.4 AI 인사이트 페이지
- AI가 분석한 코드 이슈 패턴 목록 (Mock)
- 이슈 심각도별 분류 (Critical / Warning / Info)
- 개선 제안 메시지

## 4. 비기능 요구사항
- 다크모드 기본 제공
- 페이지 전환 트랜지션
- 수치 카운터 애니메이션
- 반응형 레이아웃 (1280px 기준)

## 5. 기술 스택
| 분류 | 기술 |
|------|------|
| Frontend | Vue3, Vite, SCSS, ApexCharts, Vue Router |
| Backend | Java, Spring Boot 3, REST API |
| 데이터 | Mock JSON (실제 DB 미사용) |
| 빌드 | Maven |

## 6. 제외 범위
- 실제 GitHub API 연동
- 사용자 인증/로그인
- 실제 AI 모델 연동 (Mock 데이터로 대체)
